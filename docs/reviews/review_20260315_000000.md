## 🔴 Critical Issues

### [C-1] Object URL のメモリリーク ✅ 修正済み

**ファイル**: `src/client/components/profile/ProfileForm.tsx`
**問題**: `URL.createObjectURL()` で生成した Blob URL を `URL.revokeObjectURL()` で解放していない。ユーザーが複数回画像を選択するたびにメモリが解放されず、コンポーネントアンマウント時にも残り続ける。

**修正内容**:
- `handleFileChange` で新しい URL 生成前に旧 URL を revoke
- `useEffect` cleanup で `avatarPreviewUrl` 変更時・アンマウント時にも revoke

---

### [C-2] `onBlur` によるスキル入力の誤作動 ✅ 修正済み

**ファイル**: `src/client/components/profile/ProfileForm.tsx`
**問題**: インライン入力の `onBlur` は、ユーザーがスキルタグの「×」ボタンをクリックしたときにも発火する。

**修正内容**:
- 削除ボタン全てに `onMouseDown={e => e.preventDefault()}` を追加し、blur の誤発火を抑制
