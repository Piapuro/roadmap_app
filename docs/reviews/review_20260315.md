### [C-1] `WizardShell.tsx:46-48` — teamId が空のまま保存APIを呼べてしまう

```tsx

// ✅ 修正案

  setErrors({ form: 'チームIDが取得できませんでした。再読み込みしてください。' })
  return
}
```
